# 财大优选 - 购物车功能实现说明

## 项目概述
本项目实现了完整的购物车功能，包括前端页面、后端API服务和MySQL数据库。

## 技术栈
- **前端**: HTML5, CSS3, JavaScript (原生)
- **后端**: Spring Boot 4.0.6, Spring Data JPA
- **数据库**: MySQL 8.0+
- **构建工具**: Maven

## 功能特性
1. ✅ 添加商品到购物车
2. ✅ 查看购物车列表
3. ✅ 更新商品数量（增加/减少）
4. ✅ 删除购物车商品
5. ✅ 计算购物车总价
6. ✅ 显示购物车商品数量

## 数据库设置

### 1. 创建数据库
执行以下SQL脚本创建数据库和表结构：

```bash
# 方式1: 使用MySQL命令行
mysql -u root -p < src/main/resources/db/schema.sql

# 方式2: 在MySQL客户端中直接执行SQL文件内容
```

SQL脚本位置: `src/main/resources/db/schema.sql`

### 2. 配置数据库连接
修改 `src/main/resources/application.properties` 文件中的数据库配置：

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/caida_db?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai
spring.datasource.username=root
spring.datasource.password=你的数据库密码
```

**注意**: 请将 `spring.datasource.password` 修改为你的MySQL实际密码。

## 启动步骤

### 1. 启动后端服务

```bash
# 在项目根目录执行
mvn spring-boot:run

# 或者先编译再运行
mvn clean package
java -jar target/caida-0.0.1-SNAPSHOT.jar
```

后端服务将在 `http://localhost:8080` 启动

### 2. 打开前端页面
直接在浏览器中打开 `caida-qianduan/index.html` 文件

或者使用本地服务器（推荐）：
```bash
# 使用Python
cd caida-qianduan
python -m http.server 8000

# 使用Node.js
npx http-server caida-qianduan -p 8000
```

然后访问 `http://localhost:8000`

## API接口说明

所有API接口前缀: `http://localhost:8080/api/cart`

### 1. 添加商品到购物车
- **URL**: `/add`
- **方法**: POST
- **请求体**:
```json
{
  "userId": 1,
  "productId": 1,
  "quantity": 1
}
```
- **响应**:
```json
{
  "code": 200,
  "message": "添加成功",
  "data": { ... }
}
```

### 2. 获取购物车列表
- **URL**: `/?userId=1`
- **方法**: GET
- **响应**:
```json
{
  "code": 200,
  "message": "操作成功",
  "data": [
    {
      "id": 1,
      "userId": 1,
      "productId": 1,
      "quantity": 2,
      "totalPrice": 598.00,
      "productName": "潮色指甲油",
      "productPrice": 299.00,
      "productImage": "images/goods01.png"
    }
  ]
}
```

### 3. 更新商品数量
- **URL**: `/update`
- **方法**: PUT
- **请求体**:
```json
{
  "cartId": 1,
  "quantity": 3
}
```

### 4. 删除购物车商品
- **URL**: `/remove/{cartId}`
- **方法**: DELETE

### 5. 获取购物车数量
- **URL**: `/count?userId=1`
- **方法**: GET
- **响应**:
```json
{
  "code": 200,
  "message": "操作成功",
  "data": {
    "count": 3
  }
}
```

## 使用说明

1. **浏览商品**: 打开首页可以看到商品列表
2. **添加到购物车**: 点击商品卡片上的"加入购物车"按钮
3. **查看购物车**: 点击右上角的"购物车"按钮
4. **修改数量**: 在购物车侧边栏中，点击"+"或"-"按钮调整数量
5. **删除商品**: 点击商品右侧的"×"按钮删除
6. **结算**: 点击"去结算"按钮（目前仅显示总金额）

## 注意事项

1. 确保MySQL服务已启动
2. 数据库用户名和密码配置正确
3. 后端服务必须先于前端页面启动
4. 前端JavaScript中配置的API地址为 `http://localhost:8080/api`
5. 当前用户ID硬编码为1（在script.js中），实际项目中应从登录状态获取

## 常见问题

### Q1: 无法连接到数据库
**A**: 检查MySQL服务是否启动，以及application.properties中的配置是否正确

### Q2: 跨域问题
**A**: 后端已配置@CrossOrigin允许跨域，如仍有问题请检查浏览器控制台

### Q3: 商品图片不显示
**A**: 确保图片路径正确，图片位于 `caida-qianduan/images/` 目录下

### Q4: 端口被占用
**A**: 修改application.properties中的 `server.port` 配置
