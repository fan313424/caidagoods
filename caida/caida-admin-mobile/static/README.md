# 静态资源说明

## 图标文件

本应用使用文本表情符号作为图标，无需额外的图片文件。

如需添加自定义图标，请将以下文件放置在此目录：

- `logo.png` - 应用Logo（建议尺寸：200x200px）
- `default-avatar.png` - 默认头像（建议尺寸：100x100px）
- `default-product.png` - 默认商品图（建议尺寸：200x200px）

## TabBar图标

当前版本使用纯文本TabBar，不依赖图标文件。
如需添加图标，请在 `tabbar/` 子目录中添加：

- `home.png` / `home-active.png`
- `product.png` / `product-active.png`
- `user.png` / `user-active.png`
- `order.png` / `order-active.png`

建议图标尺寸：81x81px
