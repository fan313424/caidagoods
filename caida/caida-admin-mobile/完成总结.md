## 🚀 如何运行

### 方法一：HBuilderX（推荐）

1. **打开 HBuilderX**
   - 下载地址：https://www.dcloud.io/hbuilderx.html

2. **导入项目**
   ```
   文件 -> 导入 -> 从本地目录导入
   选择：D:\itfan\caida\caida-admin-mobile
   ```

3. **运行项目**
   - 点击工具栏 "运行" 按钮
   - 选择 "运行到浏览器" -> "Chrome"
   - 或选择 "运行到手机或模拟器"

4. **测试登录**
   - 用户名：`admin`
   - 密码：`123456`

### 方法二：命令行（需要配置）

```bash
# 进入项目目录
cd D:\itfan\caida\caida-admin-mobile

# 运行 H5
npm run dev:h5

# 运行微信小程序
npm run dev:mp-weixin
```

## 🔧 技术实现

### 前端框架
- **uni-app**: 跨平台开发框架
- **Vue.js 2.x**: 渐进式 JavaScript 框架

### 核心技术
```javascript
// 网络请求封装
utils/config.js
- request()     // 统一请求方法
- checkLogin()  // 登录状态检查
- getAdminInfo() // 获取管理员信息
- logout()      // 退出登录

// 数据存储
uni.storage
- admin_token   // JWT Token
- admin_info    // 管理员信息
```

### 样式方案
- CSS3 Flexbox 布局
- 渐变色设计
- 响应式单位（rpx）
- 公共样式库（common/uni.css）

## ⚠️ 重要说明

### 当前限制

1. **模拟数据**
   - 所有页面使用硬编码的模拟数据
   - 需要后端 API 才能实现真实数据交互

2. **图片资源**
   - 使用表情符号替代图标
   - 如需真实图片，需添加到 static 目录

3. **后端接口**
   - 已预留 API 调用代码（注释状态）
   - 需要后端实现对应接口后启用

### 需要的后端接口

详见 `项目概览.md` 中的 "后端接口需求" 章节

主要接口：
