# 财大优选管理端 - 项目概览

## 📱 项目简介

财大优选管理端是一款基于 uni-app 开发的跨平台移动应用管理系统，用于管理校园电商平台的商品、用户、员工和订单。

## ✨ 主要特性

- 🚀 **一次开发，多端运行** - 支持 H5、微信小程序、Android、iOS
- 🎨 **现代化 UI** - 渐变色设计，流畅的动画效果
- 📊 **数据可视化** - 首页数据统计面板
- 🔐 **安全登录** - Token 认证机制
- 📦 **商品管理** - 完整的 CRUD 操作
- 👥 **用户管理** - 用户状态控制
- 👔 **员工管理** - 员工信息管理
- 🛒 **订单管理** - 订单状态跟踪和发货

## 🏗️ 技术架构

## 🎯 核心功能模块

### 1. 登录模块 (`pages/login/login.vue`)
- 管理员账号密码登录
- Token 自动保存和验证
- 登录状态持久化
- 默认账号：admin / 123456

### 2. 首页模块 (`pages/index/index.vue`)
- 欢迎信息和角色显示
- 数据统计面板（用户、商品、订单、员工）
- 快捷功能入口
- 最近动态展示
- 退出登录功能

### 3. 商品管理模块
#### 商品列表 (`pages/product/list.vue`)
- 商品搜索（按名称）
- 状态筛选（全部/已上架/已下架）
- 下拉刷新和上拉加载
- 商品上下架切换
- 点击进入编辑页面

#### 添加商品 (`pages/product/add.vue`)
- 商品基本信息录入
- 分类选择器
- 图片上传（支持相册和相机）
- 表单验证
- 价格和库存管理

#### 编辑商品 (`pages/product/edit.vue`)
- 商品信息修改
- 状态切换（上架/下架）
- 图片更换
- 删除商品功能

### 4. 用户管理模块 (`pages/user/list.vue`)
- 用户搜索（用户名/手机号）
- 状态筛选（全部/正常/已禁用）
- 用户信息展示（头像、昵称、手机、邮箱）
- 启用/禁用用户
- 注册时间显示

### 5. 员工管理模块 (`pages/employee/list.vue`)
- 员工搜索（姓名/编号）
- 状态筛选
- 详细信息展示（部门、职位、联系方式）
- 启用/禁用员工
- 员工编号管理

### 6. 订单管理模块 (`pages/order/list.vue`)
- 订单搜索（订单号/用户名）
- 多维度状态筛选（待付款/待发货/已发货/已完成/已取消）
- 订单详情展示（商品信息、用户信息）
- 发货操作
- 订单总额统计

## 📊 数据流

### 登录流程
```
用户输入 → 表单验证 → API请求 → 获取Token 
→ 存储Token → 跳转首页 → 后续请求携带Token
```

### 数据请求流程
```
组件调用 → config.request() → 读取Token 
→ uni.request → 拦截响应 → 统一错误处理 
→ 返回Promise → 组件更新UI
```

### 状态管理
- 登录状态: `uni.getStorageSync('admin_token')`
- 用户信息: `uni.getStorageSync('admin_info')`
- 临时数据: 组件内部 data

## 🔐 安全机制

1. **Token 认证**
   - 登录后获取 JWT Token
   - Token 存储在本地 storage
   - 每次请求自动携带 Token

2. **路由守卫**
   - 应用启动检查登录状态
   - 未登录自动跳转登录页
   - Token 过期自动清除

3. **请求拦截**
   - 401 状态自动登出
   - 统一错误提示
   - 网络异常处理

## 🔄 后端接口需求

### 需要的 API 接口

#### 认证接口
- `POST /api/admin/login` - 管理员登录

#### 统计接口
- `GET /api/admin/stats` - 获取统计数据

#### 商品接口
- `GET /api/admin/products` - 商品列表
- `GET /api/admin/products/:id` - 商品详情
- `POST /api/admin/products` - 添加商品
- `PUT /api/admin/products/:id` - 更新商品
- `DELETE /api/admin/products/:id` - 删除商品
- `PUT /api/admin/products/:id/status` - 更新状态

#### 用户接口
- `GET /api/admin/users` - 用户列表
- `PUT /api/admin/users/:id/status` - 更新用户状态

#### 员工接口
- `GET /api/admin/employees` - 员工列表
- `PUT /api/admin/employees/:id/status` - 更新员工状态

#### 订单接口
- `GET /api/admin/orders` - 订单列表
- `PUT /api/admin/orders/:id/ship` - 订单发货

#### 上传接口
- `POST /api/admin/upload` - 图片上传
