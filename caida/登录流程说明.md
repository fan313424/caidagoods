# 登录流程说明

## 功能概述

已实现完整的登录验证流程，用户必须先登录才能访问主页面。

## 登录流程

### 1. 访问首页（index.html）
- 自动检查登录状态
- 未登录：自动跳转到登录页（login.html）
- 已登录：正常显示页面内容

### 2. 登录页面（login.html）
- 输入用户名和密码
- 勾选"记住我"可将登录信息保存到localStorage（关闭浏览器后仍保留）
- 不勾选则保存到sessionStorage（关闭浏览器后清除）
- 登录成功后跳转到首页
- 已登录用户访问登录页会自动跳转到首页

### 3. 注册页面（register.html）
- 新用户注册
- 注册成功后跳转到登录页
- 已登录用户访问注册页会自动跳转到首页

### 4. 退出登录
- 首页顶部导航栏显示"欢迎，用户名"
- 点击"退出登录"清除登录信息
- 自动跳转到登录页

## 技术实现

### 登录状态存储
```javascript
// 登录成功后保存用户信息
localStorage.setItem('userInfo', JSON.stringify(result.data));
localStorage.setItem('userId', result.data.id);

// 或使用sessionStorage（不勾选"记住我"时）
sessionStorage.setItem('userInfo', JSON.stringify(result.data));
sessionStorage.setItem('userId', result.data.id);
```

### 登录验证
```javascript
// 在script.js中检查登录状态
const userInfo = localStorage.getItem('userInfo') || sessionStorage.getItem('userInfo');
const userId = localStorage.getItem('userId') || sessionStorage.getItem('userId');

if (!userInfo || !userId) {
    window.location.href = 'login.html';
    return;
}
```

### 用户ID使用
```javascript
// 从登录状态获取用户ID，用于购物车等操作
const CURRENT_USER_ID = parseInt(userId);
```

## 使用说明

### 首次使用
1. 访问 http://localhost（或 http://localhost:80）
2. 自动跳转到登录页
3. 点击"立即注册"创建账号
4. 注册成功后跳转到登录页
5. 输入用户名和密码登录
6. 登录成功后跳转到首页

### 后续使用
1. 访问 http://localhost
2. 如果之前勾选了"记住我"，直接进入首页
3. 如果未勾选"记住我"，关闭浏览器后需要重新登录

### 退出登录
1. 点击首页顶部导航栏的"退出登录"
2. 清除登录信息
3. 自动跳转到登录页

## 注意事项

1. **后端服务必须启动**：确保Spring Boot后端运行在 http://localhost:8080
2. **Nginx必须启动**：确保Nginx正在运行并正确配置
3. **数据库连接正常**：确保MySQL数据库连接正常
4. **用户数据**：首次使用需要先注册账号

## 测试账号

如果没有测试账号，可以通过以下方式创建：

### 方式1：使用注册页面
访问 http://localhost/register.html 进行注册

### 方式2：使用API注册
```bash
curl -X POST http://localhost/api/user/register \
  -H "Content-Type: application/json" \
  -d "{\"username\":\"testuser\",\"password\":\"test123456\",\"nickname\":\"测试用户\"}"
```

### 方式3：直接操作数据库
参考 init_database.sql 文件中的表结构，插入测试用户数据
