## 🔧 常见问题解决

### 问题1：后端启动失败 - 端口被占用
**错误信息**: `Port 8080 is already in use`

**解决方案**:
修改 `application.yml`:
```yaml
server:
  port: 8081  # 改为其他端口
```

同时修改前端 `script.js` 中的 API_BASE_URL:
```javascript
const API_BASE_URL = 'http://localhost:8081/api';
```

### 问题2：数据库连接失败
**错误信息**: `Access denied for user 'root'@'localhost'`

**解决方案**:
1. 检查 MySQL 服务是否启动
2. 确认用户名密码正确
3. 尝试用命令行登录 MySQL 验证：
   ```bash
   mysql -u root -p
   ```

### 问题3：前端无法连接后端
**错误信息**: `Failed to fetch` 或 CORS 错误

**解决方案**:
1. 确认后端服务正在运行
2. 检查浏览器控制台是否有错误
3. 确认 API_BASE_URL 配置正确
4. 后端已配置跨域支持，理论上不会有CORS问题

### 问题4：Maven 依赖下载慢
**解决方案**:
配置阿里云镜像，编辑 Maven 的 `settings.xml`:
```xml
<mirrors>
  <mirror>
    <id>aliyun</id>
    <mirrorOf>central</mirrorOf>
    <name>Aliyun Maven</name>
    <url>https://maven.aliyun.com/repository/public</url>
  </mirror>
</mirrors>
```

## 📊 API 测试示例

### 使用 curl 测试

```bash
# 1. 获取商品列表
curl http://localhost:8080/api/product/list

# 2. 添加商品到购物车
curl -X POST http://localhost:8080/api/cart/add \
  -H "Content-Type: application/json" \
  -d '{"userId":1,"productId":1,"quantity":1}'

# 3. 查看购物车
curl http://localhost:8080/api/cart?userId=1

# 4. 更新数量
curl -X PUT http://localhost:8080/api/cart/update \
  -H "Content-Type: application/json" \
  -d '{"cartId":1,"quantity":3}'

# 5. 删除商品
curl -X DELETE http://localhost:8080/api/cart/remove/1

# 6. 获取购物车数量
curl http://localhost:8080/api/cart/count?userId=1
```

导入以下集合进行测试：
- 创建新 Collection: "财大优选API"
- 添加上述6个请求
