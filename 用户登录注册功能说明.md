# 用户登录注册功能说明

## 功能概述

本项目已完成完整的用户登录和注册功能，包括前端页面、后端API和数据库支持。

## 技术栈

- **前端**: HTML5, CSS3, JavaScript (原生)
- **后端**: Spring Boot, Spring Data JPA
- **数据库**: MySQL 8.0+

## 功能特性

### 1. 用户注册
- ✅ 用户名注册（4-20位字符，只能包含字母、数字和下划线）
- ✅ 密码注册（6-20位字符，必须包含字母和数字）
- ✅ 昵称（选填）
- ✅ 邮箱（选填）
- ✅ 手机号（选填）
- ✅ 用户名实时检查
- ✅ 表单验证
- ✅ 用户协议确认

### 2. 用户登录
- ✅ 用户名密码登录
- ✅ 记住我功能（使用localStorage/sessionStorage）
- ✅ 登录状态保持
- ✅ 错误提示

### 3. 数据库设计
- ✅ 用户表（user）
- ✅ 唯一索引（username）
- ✅ 状态管理（正常/禁用）
- ✅ 时间戳（创建时间/更新时间）

## API接口说明

### 1. 用户注册

**接口**: `POST /api/user/register`

**请求体**:
```json
{
  "username": "testuser",
  "password": "test123",
  "nickname": "测试用户",
  "email": "test@example.com",
  "phone": "13800138000"
}
```

**响应**:
```json
{
  "code": 200,
  "message": "注册成功",
  "data": {
    "id": 1,
    "username": "testuser",
    "nickname": "测试用户",
    "email": "test@example.com",
    "phone": "13800138000"
  }
}
```

### 2. 用户登录

**接口**: `POST /api/user/login`

**请求体**:
```json
{
  "username": "admin",
  "password": "123456"
}
```

**响应**:
```json
{
  "code": 200,
  "message": "登录成功",
  "data": {
    "id": 1,
    "username": "admin",
    "nickname": "管理员",
    "email": "admin@caida.edu.cn",
    "phone": "13800138000"
  }
}
```

### 3. 获取用户信息

**接口**: `GET /api/user/{id}`

**响应**:
```json
{
  "code": 200,
  "message": "操作成功",
  "data": {
    "id": 1,
    "username": "admin",
    "nickname": "管理员",
    "email": "admin@caida.edu.cn",
    "phone": "13800138000",
    "status": 1
  }
}
```

### 4. 检查用户名是否存在

**接口**: `GET /api/user/check-username?username=testuser`

**响应**:
```json
{
  "code": 200,
  "message": "操作成功",
  "data": {
    "exists": true
  }
}
```

```sql
-- 用户表
CREATE TABLE user (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) NOT NULL UNIQUE,
    password VARCHAR(100) NOT NULL,
    email VARCHAR(50),
    phone VARCHAR(20),
    nickname VARCHAR(50),

    update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

## 使用示例

### 注册用户

1. 打开 `register.html`
2. 填写用户名（如：newuser）
3. 填写密码（如：pass123）
4. 确认密码
5. 可选填写昵称、邮箱、手机号
6. 勾选同意用户协议
7. 点击"注册"按钮

### 登录用户

1. 打开 `login.html`
2. 输入用户名（如：admin）
3. 输入密码（如：123456）
4. 可选勾选"记住我"
5. 点击"登录"按钮
6. 登录成功后自动跳转到首页

## 安全说明

### 当前实现
- 密码以明文形式存储（仅用于学习和演示）
- 无JWT token认证
- 无密码加密

2. **Token认证**: 实现JWT token机制
3. **HTTPS**: 使用HTTPS传输
4. **验证码**: 添加图形验证码或短信验证码
5. **限流**: 防止暴力破解
6. **会话管理**: 实现安全的会话管理

## 常见问题

### Q1: 注册时提示"用户名已存在"
**A**: 尝试使用其他用户名，或检查数据库中是否已有该用户

### Q2: 登录时提示"用户名或密码错误"
**A**: 
1. 检查用户名和密码是否正确
2. 确认数据库中是否有该用户
3. 使用测试账号：admin/123456 或 test/123456

### Q3: 无法连接到后端API
**A**: 
1. 确认后端服务已启动
2. 检查控制台是否有错误信息
3. 确认数据库连接配置正确

### Q4: 跨域问题
**A**: 后端已配置 `@CrossOrigin(origins = "*")`，理论上不会有跨域问题
