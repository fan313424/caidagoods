# 登录验证测试指南

## 📋 测试步骤

### 1. 启动服务

#### 启动后端
```bash
# 方式1：使用批处理文件
双击运行 "启动后端.bat"

# 方式2：使用Maven命令
mvn spring-boot:run
```

#### 启动Nginx
```bash
# Windows
nginx

# 或重新加载配置
nginx -s reload
```

### 2. 测试登录功能

#### 方式1：使用测试页面（推荐）
访问：http://localhost/test-login.html

**测试步骤：**
1. 点击"检查登录状态" - 应该显示"未登录"
2. 输入用户名和密码（默认：test / 123456）
3. 点击"登录" - 应该显示"登录成功"
4. 再次点击"检查登录状态" - 应该显示已登录信息
5. 点击"跳转到首页" - 应该能正常访问首页

#### 方式2：直接访问首页
访问：http://localhost/index.html

**预期行为：**
- 如果未登录，会自动跳转到登录页
- 如果已登录，正常显示首页内容

#### 方式3：通过登录页
访问：http://localhost/login.html

**测试步骤：**
1. 如果已登录，会自动跳转到首页
2. 如果未登录，显示登录表单
3. 输入用户名和密码
4. 勾选/不勾选"记住我"
5. 点击登录
6. 登录成功后跳转到首页

### 3. 测试退出登录

访问首页后，点击顶部导航栏的"退出登录"按钮，应该：
1. 清除登录信息
2. 跳转到登录页
3. 再次访问首页会跳转到登录页

## 🔍 问题排查

### 问题1：访问首页没有跳转到登录页

**可能原因：**
- 浏览器缓存了旧的 script.js
- localStorage/sessionStorage 中有旧的登录信息

**解决方法：**
1. 清除浏览器缓存（Ctrl + F5 强制刷新）
2. 打开浏览器开发者工具（F12）
3. 进入 Application（应用）标签
4. 清除 Local Storage 和 Session Storage
5. 刷新页面

### 问题2：登录后没有跳转到首页

**可能原因：**
- 后端服务未启动
- Nginx配置错误
- API路径错误

**解决方法：**
1. 检查后端是否运行：访问 http://localhost:8080/api/product/list
2. 检查Nginx配置：确保 proxy_pass 正确
3. 查看浏览器控制台是否有错误信息

### 问题3：登录成功但首页仍然跳转到登录页

**可能原因：**
- localStorage/sessionStorage 保存失败
- 读取时数据丢失

**解决方法：**
1. 使用测试页面验证登录信息是否正确保存
2. 检查浏览器是否禁用了存储功能
3. 查看控制台是否有错误信息

## 🛠️ 调试技巧

### 1. 查看存储信息
打开浏览器开发者工具（F12）：
- Application（应用） → Storage（存储） → Local Storage
- Application（应用） → Storage（存储） → Session Storage

应该看到：
- `userInfo`: 用户信息JSON字符串
- `userId`: 用户ID

### 2. 查看网络请求
打开浏览器开发者工具（F12）：
- Network（网络）标签
- 查看登录请求是否成功
- 查看返回的数据格式

### 3. 查看控制台错误
打开浏览器开发者工具（F12）：
- Console（控制台）标签
- 查看是否有JavaScript错误

## 🔄 完整流程测试

1. **清除所有缓存和存储**
   - 清除浏览器缓存
   - 清除 Local Storage 和 Session Storage

2. **访问首页**
   - 访问 http://localhost/index.html
   - 应该自动跳转到 http://localhost/login.html

3. **注册新用户**
   - 点击"立即注册"
   - 填写注册信息
   - 注册成功后跳转到登录页

4. **登录**
   - 输入用户名和密码
   - 勾选"记住我"
   - 点击登录
   - 登录成功后跳转到首页

5. **验证登录状态**
   - 首页显示"欢迎，用户名"
   - 显示"退出登录"按钮
   - 购物车功能正常使用

6. **退出登录**
   - 点击"退出登录"
   - 自动跳转到登录页
   - 再次访问首页会跳转到登录页

7. **测试记住我功能**
   - 重新登录（勾选"记住我"）
   - 关闭浏览器
   - 重新打开浏览器访问首页
   - 应该仍然保持登录状态

8. **测试不记住我**
   - 退出登录
   - 重新登录（不勾选"记住我"）
   - 关闭浏览器
   - 重新打开浏览器访问首页
   - 应该跳转到登录页
