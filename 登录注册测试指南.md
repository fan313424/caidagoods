## ⚠️ 常见问题排查

### 问题1: 注册失败，提示"用户名已存在"

**解决方案**:
```sql
-- 查看已存在的用户名
SELECT username FROM user;

-- 如果不需要某个用户，可以删除
DELETE FROM user WHERE username = '不需要的用户名';
```

### 问题2: 登录失败，提示"用户名或密码错误"

**检查步骤**:
1. 确认用户名拼写正确
2. 确认密码正确（注意大小写）
3. 检查数据库中用户是否存在：
   ```sql
   SELECT * FROM user WHERE username = '你的用户名';
   ```
4. 检查用户状态是否为1（正常）：
   ```sql
   UPDATE user SET status = 1 WHERE username = '你的用户名';
   ```

### 问题3: 前端无法连接后端

**检查步骤**:
1. 确认后端服务正在运行
2. 访问 `http://localhost:8080/api/user/check-username?username=test` 看是否有响应
3. 检查浏览器控制台是否有CORS错误
4. 确认防火墙没有阻止8080端口

### 问题4: 数据库连接失败

**解决方案**:
1. 检查MySQL服务是否启动
2. 确认 `application.yml` 中的数据库配置：
   ```yaml
   spring:
     datasource:
       url: jdbc:mysql://localhost:3306/caida_db?...
       username: root
       password: 你的密码
   ```
3. 测试数据库连接：
   ```bash
   mysql -u root -p caida_db
   ```
